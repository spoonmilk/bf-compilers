# bf-compilers
I think a fun way to learn a language is to write a program you know well in it. One such program is a BrainFuck compiler. I will write a BrainFuck compiler in every language I learn and put it here. No AI, mkay?

I was thinking about how best to make this as cross-platform as possible. One manner would be to make it as un cross-platform as possible and target RISC-V
so that none of you nerds could use this without an emulator. Another would be to target WebAssembly so all of you nerds could use it, just in a browser.
I have decided, though, that the *absolute best* way to make it cross-platform would be to *write each compiler for a different platform.* Surely,
as I learn every programming language in the world, I'll be able to then cover your platform.
